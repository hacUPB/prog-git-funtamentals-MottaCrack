# Crear un Repositorio Remoto

**1** Iniciar sesion en tu cuenta de GitHub.

**2** Seleccionar la opcion "New Repository" que aparece arriba a la derecha.

**3** Ponerle nombre al repositorio y darle al boton verde "Create New Repository".

**4** Navegar en los directorios locales usando la consola hasta llegar a la carpeta donde queremos que se cree la copia del repositorio remoto.

**5** Usar el comando "Git Clone" con el URL del repositorio para crear una copia de este en tu maquina local. (El URL se encuentra en el boton verde "Code")

**6** Despues de hacer cambios en la copia local debes hacer un commit y usar el comando "git push" para actualizar el repositorio remoto con los ultimos cambios realizados al repositorio local.
